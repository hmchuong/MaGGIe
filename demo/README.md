# Demo

## Install
Please install additional libraries by running:
```bash
pip install -r requirements.txt
```

To work on video, you need to install `ffmpeg`
```
sudo apt-get install ffmpeg
```

## Set up XMem
```bash
git clone https://github.com/hkchengrex/XMem.git
cd XMem
pip install -r requirements.txt
```

## Running
To run demo:
```bash
gradio app.py
```
